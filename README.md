# Algoritmos e Estrutura de Dados — UPE

Projeto de visualização de algoritmos de ordenação (Bubble Sort e Merge Sort) desenvolvido para o seminário de Estrutura de Dados.

## Funcionalidades
- Visualização passo a passo do Bubble Sort.
- Visualização passo a passo do Merge Sort.
- Comparação de complexidade entre algoritmos.
- Roteiro de apresentação integrado.
- Slides de apoio.

## Como Executar Localmente

1. Instale as dependências:
   ```bash
   npm install
   ```

2. Configure o arquivo `.env`:
   ```env
   VITE_UPE_APP_ID=dummy-id
   VITE_UPE_FUNCTIONS_VERSION=v1
   VITE_UPE_APP_BASE_URL=http://localhost:5173
   ```

3. Inicie o servidor de desenvolvimento:
   ```bash
   npm run dev
   ```
